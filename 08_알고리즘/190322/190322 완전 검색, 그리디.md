### 조합 생성

```python
def nCr(n, r):
    if r == 0 or n == r:
        return 1
    return nCr(n - 1, r - 1) + nCr(n - 1, r)

memo = [[0] * 100 for _ in range(100)]
def nCr(n, r):
    if r == 0 or n == r:
        return 1
    if memo[n][r]:
        return memo[n][r]
    
    memo[n][r] = nCr(n - 1, r - 1) + nCr(n - 1, r)
    return memo[n][r]
```
